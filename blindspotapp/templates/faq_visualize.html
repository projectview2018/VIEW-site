Each table row has an individual data points for vehicles entered into our database. Body class and weight class are convenient ways to separate vehicles by purpose - body class categories include passenger vehicles, SUVs, minivans, vans, pickup trucks, commercial trucks, and buses. Both of these can be manually entered, but are generally pulled using VINs from entries.

The VIEW app is a crowdsourced database where multiple different users can add data and can add multiple entries to further increase confidence in the results. Although most drivers will adjust their seats to end up at the same rough eye point location, all VIEW app outputs will be specific to drivers doing the input process. Therefore, some partners will want to enter vehicles multiple times to gain multiple perspectives on the blind zones.
